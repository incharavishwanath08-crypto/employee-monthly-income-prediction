# Employee Monthly Income Prediction Using Machine Learning

## Project Overview

This project focuses on predicting employee monthly income using machine learning regression techniques. The objective is to analyze employee-related factors and build predictive models that accurately estimate monthly income. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model building, and performance evaluation.

## Problem Statement

Organizations often need to understand the factors influencing employee compensation. This project aims to develop a predictive model that estimates an employee's monthly income based on various demographic, educational, and professional attributes.

## Dataset

The dataset contains employee-related information such as:

- Age
- Gender
- Education
- Department
- Job Role
- Work Experience
- Marital Status
- Other employee attributes

**Target Variable:**
- Monthly Income

## Project Workflow

### 1. Data Preprocessing
- Handling missing values
- Data cleaning
- Encoding categorical variables
- Feature selection

### 2. Exploratory Data Analysis (EDA)
- Distribution analysis
- Correlation analysis
- Feature relationship visualization
- Identification of income-influencing factors

### 3. Model Development
The following regression algorithms were implemented and evaluated:

- Linear Regression
- Multiple Linear Regression
- Polynomial Regression
- Ridge Regression
- Lasso Regression
- Decision Tree Regression

### 4. Model Evaluation
Models were evaluated using:

- R² Score
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Percentage Error (MAPE)
